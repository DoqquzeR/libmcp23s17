Waits for an interrupt from the mcp23s17 or until timeout is reached. \begin{DoxyNote}{Note}
This method does N\-O\-T reset the interrupt -\/ which is done automatically for you by reading the input state register. Calling this method twice in a row without reading the input register will cause it to always wait for your timeout value, regardless of button presses. To avoid this, read the input register after every call to this method.
\end{DoxyNote}
